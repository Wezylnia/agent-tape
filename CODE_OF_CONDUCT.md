# Code of Conduct

AgentTape follows a simple contributor standard: be direct, respectful, and focused on the work.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different experience levels and communication styles.
- Keep technical disagreement specific and evidence-based.
- Accept maintainers' scope decisions when they are made to protect project focus.
- Report security-sensitive concerns privately through the process in `SECURITY.md`.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Publishing private information without explicit permission.
- Repeated off-topic arguments after a maintainer asks to move on.
- Using issues or pull requests for spam, promotion, or unrelated support.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; decline pull requests; or block participants when needed to keep the project healthy.

For concerns, open a private security advisory when security-related, or contact the maintainer through the GitHub profile for conduct issues.
